OK OK so theres many things to do with Dialogue.

The character images are the same as a character name, so you can use all the images in this folder. You can also add more portraits with the mods folder. You can create dialogue like so:

:dave:Hi!

You can look in data/dialogue/portraits to change aliasing and the sound.

There are currently 14 events that can happen in dialogue. This includes:

setfont_code:none:You can't see my text!
setfont_normal:none:You can see my text!
black:none:Instant black.
no_black:none:Instant no more black.
to_black:none:Transition into darkness.
off_black:none:Transition out of darkness.
droptext_green:bambi:I'm green now!
droptext_blue:dave:I'm blue now!
droptext_white:gfcasual:White people tomorrow morning
showblitz:3ddave:Shows the Blitz cutscene image! Wow!
:noblitz:3ddave:No longer!
:slowDialogueSpeed:bambimad:I'm slow.
:fastDialogueSpeed:dave:I'm fast!
:normalDialogueSpeed:bf:Hey

Only one event can happen at a time.
